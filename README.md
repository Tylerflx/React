# React for Beginner
Learn and practice Reactjs by Tyler Nguyen


#### 🏃🏼‍♂️ How to run program 💨
1. Clone repository
    ```bash
    git clone https://github.com/Tylerflx/React.git
    ```
2. Change directory to folder
    >Ex: cd changing-state
3. Open Terminal/Cmd, run
    ```bash
    npm install
    ```
4. Once done, run
    ```bash
    npm start
    ```
