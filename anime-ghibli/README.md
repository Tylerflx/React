## About project

The project is based on working with API data. API address - https://ghibliapi.herokuapp.com/.

## Instalation

```bash
1. Clone project
2. Change Directory to project
    cd anime-ghibli
3. Unload packages
    npm install
4. Run project
    npm start
```
